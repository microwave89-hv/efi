Esta atualização fornece correções de estabilidade para Thunderbolt e Modo de Disco de Destino.

Quando seu computador reiniciar, uma tela cinza aparecerá com uma barra de estado para indicar o progresso da atualização. Não perturbe ou desligue a alimentação do seu computador durante essa atualização. Observação: Seu computador deve estar conectado a uma fonte de energia AC para que a atualização seja instalada.
