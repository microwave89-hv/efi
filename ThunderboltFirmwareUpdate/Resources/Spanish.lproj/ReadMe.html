Esta actualización resuelve problemas de estabilidad de Thunderbolt y la modalidad de disco de destino.

Al reiniciar el ordenador, aparecerá una pantalla gris con una barra de estado que indica el progreso de la actualización. No realice modificaciones ni apague el ordenador durante el proceso de actualización. Nota: Para instalar esta actualización, el ordenador debe estar enchufado a la corriente.
