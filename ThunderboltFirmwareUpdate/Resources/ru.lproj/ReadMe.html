Данное обновление содержит исправления, повышающие стабильность Thunderbolt и режима внешнего диска.

При перезагрузке компьютера появится серый экран со строкой состояния, показывающей ход выполнения обновления. Не прерывайте этот процесс и не выключайте питание компьютера во время обновления. Примечание. Для установки обновления компьютер должен быть подключен к источнику переменного тока.
