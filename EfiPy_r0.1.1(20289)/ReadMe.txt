    Author: EfiPy Core (efipy.core@gmail.com)

2. Description

    EfiPy provides minimum C(CPython) code for interface between Python and
    EDK II kernel.

    EfiPy includes these parts
    1. _EfiPy: EDK II minimized C code.
    2. EfiPy\MdePkg: UEFI base data type, spec defined structure and protocol.
    3. EfiPy\ShellPkg: UEFI shell defined interface.
    4. EfiPyShellPkg: Simple UEFI shell program coded with EfiPy library to prove EfiPy workable.
    5. Analyzer: EDKII protocol flow analyzer.
